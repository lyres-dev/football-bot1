## 📋 Установка (5 минут)

### 1. Скопируй файлы в папку

```
football_bot/
├── bot.py
├── api_client.py
├── analyzer.py
├── requirements.txt
└── .env
```

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. Получи API-ключи (все бесплатно!)

#### Telegram Bot Token
1. Открой @BotFather в Telegram
2. Отправь /newbot
3. Придумай имя и username
4. Скопируй токен вида 1234567890:ABCdef...

#### The Odds API Key
1. Зайди на https://the-odds-api.com
2. Нажми "Get API Key" и зарегистрируйся
3. Скопируй ключ из личного кабинета
4. Бесплатно: 500 запросов/месяц

#### Google Gemini API Key (БЕСПЛАТНО!)
1. Зайди на https://aistudio.google.com
2. Войди через Google аккаунт
3. Нажми "Get API key" -> "Create API key"
4. Скопируй ключ
5. Бесплатно: 1500 запросов/день — более чем достаточно!

### 4. Создай файл .env

Переименуй .env.example в .env и заполни:

```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
ODDS_API_KEY=abc123def456ghi789
GEMINI_API_KEY=AIzaSy...
```

### 5. Запусти!

```bash
python bot.py
```

## Как работает прогноз

1. Бот запрашивает коэффициенты букмекеров через The Odds API
2. Переводит их в вероятности (1/коэффициент x 100%)
3. Отправляет данные в Gemini 2.0 Flash с запросом на анализ
4. Gemini генерирует развёрнутый прогноз на русском языке
